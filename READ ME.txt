## Local Setup Instructions

Follow these exact steps sequentially to get your local development environment up and running:

### 1. Clone the Project Repository
```bash
git clone https://github.com
cd wills-news-application
```

### 2. Configure a Virtual Environment
```bash
# Create the isolated environment
python -m venv .venv

# Activate it (Windows)
.venv\Scripts\activate

# Activate it (Mac/Linux)
source .venv/bin/activate
```

### 3. Install Dependencies
Make sure your `.venv` is active, then install all project requirements:
```bash
pip install -r requirements.txt
```

### 4. Apply Database Migrations
This configures your local database schemas and automatically seeds your user permission roles:
```bash
python manage.py migrate
```

### 5. Launch the Local Server
```bash
python manage.py runserver
```
Once launched, open your web browser and navigate to: `http://127.0.0`
